# AquaAI Local Development Setup

Follow these steps to get AquaAI running on your local machine.

## Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- Expo CLI (`npm install -g expo-cli`)
- Expo Go app installed on your physical iOS/Android device (or a configured simulator)

## 1. Installation
Clone the repository and install dependencies:
```bash
git clone [https://github.com/yourusername/aquaai.git](https://github.com/yourusername/aquaai.git)
cd aquaai
npm install --legacy-peer-deps
